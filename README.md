# Anonymous Blog üìù

A simple full-stack **Anonymous Blogging Application** where users can create, edit, and delete blog posts without authentication.  
Built to practice CRUD operations, clean UI, and API integration.
## Features

- Create blog posts anonymously
- Edit existing posts
- Delete posts
- View all posts in a clean, modern UI
- Automatic date & ID generation
- Responsive and modern design (Inter font)


## Tech Stack

### Frontend
- React
- React Router
- Axios
- Vite
- CSS (Global styling)

### Backend
- Node.js
- Express
- PostgreSQL

## Database Schema

```sql
CREATE TABLE blog (
  id INT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  author VARCHAR(100) NOT NULL,
  date TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

```
## Run Locally

Clone the project

```bash
  git clone https://github.com/Bhavykun/Anonymous-Blog.git
```

Go to the project directory

```bash
  cd Anonymous-Blog
```

Install dependencies

```bash
  npm install
```

Start the frontend

```bash
  npm run start
```

Start the API

```bash
  node APIdb.js
```
Make sure your backend server and PostgreSQL database are running and correctly configured.
